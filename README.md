# 🛒 Ecommerce Website (Crocus Project)

## 📌 Project Description

This is an Ecommerce website developed using ASP.NET. Users can browse products, add them to the cart, and place orders online.

## 🚀 Features

* User Registration and Login
* Product Listing
* Add to Cart
* Order Placement
* Admin Dashboard

## 🛠️ Technologies Used

* ASP.NET
* C#
* SQL Server
* HTML, CSS, JavaScript, Boostrap

## 📂 Project Structure

* Pages (.aspx)
* Code-behind files (.cs)
* Database
* CSS and JavaScript files

## ▶️ How to Run

1. Open the project in Visual Studio
2. Connect the database
3. Press F5 to run the project
